Runing the project:
1. go to ATMMAP \BS-PMC-2023-Team26\ATMMAP
2. run python manage.py makemigrations
3. run python manage.py migrate
4. run python manage.py runserver
5. go to \BS-PMC-2023-Team26\ATMMAP\frontend
6. run npm run dev

In case of db desync run: python manage.py migrate --run-syncdb
